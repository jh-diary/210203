       css의 개념
      html- 문서의 내용
      css- 문서의 스타일
      javascript- 문서의 동작

      css의 중요한 모듈은 selectors, box model, backgrounds and borders, text effects, 2D/3D transformations, animations, multiple column layout, user interface 등이 있다.
      html과 css를 분리하는 것은 동일한 css style 태그를 사용하여 관리가쉽고, 일관성을 유지하고, 시간이 단축된다.
      *여러개의 css가 충돌할 때는 우선순위에 따라서 해결된다.
      css 문법은
      선택자{
        속성(property):값(value);
      }의 방식으로 작성한다.
      css의 위치는 head태그 안에 작성하거나 css파일을 따로 만들어 link태그를 이용해서 적용한다.
    

    

      선택자
      선택자의 종류 중 가장 많이 사용하는 6가지 종류
      
    
        
  • type selectors
    • - html 요소 이름을 사용하는 것 p요소를 사용하면 p라고 쓰는 것 ex) p{color:red;}
        
  • universal selectors
    • - *기호로 표시되며 페이지 안 모든 요소를 선택할 때 사용된다(주로 공통 속성 지정할 때 사용) ex) *{color:red;}
        
  • id selectors
    • - html에 id를 부여하고 사용, 제일 많이 사용되는 선택자 ex) #id{color:red;) h1#id{color:red;}
        
  • class selectors
    • - id selector는 하나의 요소 만을 선택할 수 있지만 class selectors는 여러개의 요소를 함께 grouping 시킬 수 있다. ex) .class{color:red;} h1.class{color:red;}}
        
  • pseudo class
    •  :사용, 의사 클래스라고 하고 이는 클래스가 정의된 것처럼 간주한다는 의미이다. ex) a:link라고 하면 a태그에 link가 선언된 것처럼 생각하고 선택자를 만드는 것이다.
                              ex) a:link{color:blue;}  /*아직 방문되지않은 링크는 파란색*/
                                  a:visited{color:red;} /*방문된 링크는 red*/
                                  a:visited{color:green;} /*현재 위치한 페이지는 green*/
        
  • 속성 선택자
    • - 특정한 속성을 가지는 요소 선택 ex) h1[title]{color:red;}- title 속성을 가진h1요소를 선택
                                                              p[class="example"]- class속성이 example인 p요소 선택
      

      type, id, class selectors가 제일 중요!
    

    

      css를 삽입하는 방법
      
    
        
  1. external style sheet
    2. - 각 html 페이지는 link 태그를 이요해서 style 파일을 연결해야한다. link태그는 head태그안에 써야함. 확장자는 .css이다.
        
  2. internal style sheet
    3. - html 안에 style태그를 지정하는 방법 이것도 style태그를 head태그안에 쓴다.
        
  3. inline style sheet
    4. - 각각의 요소마다 스타일을 지정한다. 이 방법은 비추... 정말 필요한 경우만 사용
        

        css 태그의 우선순위는 1.inline style 2. internal style 3. external style 4. 웹브라우저 디폴트값
        부모태그 및 조상태그에 지정된 css속성은 자식태그 및 후손태그까지 전부 적용이 되기때문에 자식 및 후손태그의 스타일을 지정하려면 개별적으로 지정해준다.
        또한 class selectors와 id selectors를 잘 활용하는 것은 중요하다!
